Threat Modeling: Aplikasi Antrian

Pendahuluan

Threat modeling adalah metode terstruktur untuk mengenali, menilai, dan memitigasi potensi ancaman keamanan pada sebuah sistem. Tujuannya adalah menemukan titik-titik rentan sejak awal—seperti autentikasi tidak aman, kebocoran data, atau kemungkinan eskalasi hak akses—lalu menyusun langkah pencegahan baik secara teknis maupun prosedural. Pada artikel ini, proses threat modeling diterapkan pada aplikasi antrian berbasis PHP + MySQL dari repository galihap76/antrian sebagai contoh studi kasus.

Persiapan

1. Instalasi Server

  1. Unduh XAMPP melalui situs resmi: https://www.apachefriends.org/
  2. Jalankan layanan Apache dan MySQL dari XAMPP Control Panel.
  3. Cek apakah server aktif dengan membuka: http://localhost. Jika dashboard XAMPP muncul, berarti server sudah berjalan.

2. Instalasi Aplikasi Antrian

  1. Buka terminal dan masuk ke direktori web server (XAMPP/htdocs).

3. Import Database

  1. Masuk ke phpMyAdmin dan buat database baru bernama antrian.
  2. Import file SQL yang berada di: antrian/assets/db_antrian.sql.
  3. Jika proses impor berhasil, database siap digunakan.
